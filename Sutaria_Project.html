Computer Science Principles - Scratch Project
Semptember 2019

The objective of this project was to create a game or a story using Scratch. My partner and I decided to make a hide and seek game. The main goal of the game was to find all the characters in each of the levels with the time given. Some features of the game are moving up, down, left, or right to go to different backdrops. Another feature was adding a point to your score everytime you find a character that is hiding.This game has 3 different levels. The first level is castle themed with 30 seconds to find the first character. The second level is beach themed which adds another character and the time is reduced to 20 seconds. The last level has three different characters with a playground theme with only 10 seconds to find all of the characters. My role is this project was to create levels 1 and 3. I was also responsible for making the base of the game and sharing it with my partner.
